# 🎹 QBirdSynth

A modern reimagining of the classic QiBrd synthesizer app — rebuilt from scratch with:

- **🌐 Web Version** — SolidJS + Web Audio API, zero-install browser synth
- **📱 Android Version** — Kotlin + Jetpack Compose + Csound 6.18

**80s retro-futurism design**: Purple · Orange · Dark Grey · Black with CRT scanlines and neon glow effects.
